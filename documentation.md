# Factrack
Een turret die een gezicht volgt met behulp van een (laptop) camera.

Progressie:
Week 1: Ideeen bedenken
Week 2: Gekomen op een turret bestuurdt met een ps3 controller
Week 3: Erachter gekomen dat de ps3 controller niet al te soepel werkt en
daarom naar een camera input verandert
Week 4: Twee scripts  maken en testen met 1 servo om horizontaal te tracken
Script 1: Python script dat camera x,y cords doorstuurt naar de arduino
Script 2: Arduino script dat deze x en y cords binnenkrijgt en het in graden veranderd voor de servo
Week 5: Script verbeteren en Y toevoegen zodat hij ook verticale bewegingen kan tracken
Week 6: Helpen met 3D model maken zodat er een platform is voor een attachment zoals bijvoorbeeld een laser
Week 7: Zelfde als week hiervoor
